My work is focused on creating a cohesive ecosystem of open-source software that respects user freedom, from desktop to mobile.

Cerberus Linux for Mobile →

A revolutionary project to bring a true Linux environment to Android hardware. By leveraging the AngusOS HAL, Cerberus delivers unparalleled control, breaking the chains of the standard Android environment.

 AngusOS Foundation →

The essential Android distribution maintained as a clean, stable hardware interface layer, making the power of Cerberus Linux possible on a wide range of mobile devices.

A privacy-focused browser fork that empowers users with greater privacy controls and a more efficient browsing experience, free from unwanted telemetry and bloat.
