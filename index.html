About Me

Salutations. I am Natalie Spiva, though widely known in digital circles as Sprungles. My work is dedicated to architecting systems that empower the user, free from the constraints of increasingly locked-down, proprietary ecosystems.

My flagship project is ArttulOS, a complete Linux distribution detailed below. This project is the culmination of my efforts to build robust, open-source platforms for both desktop and mobile hardware, giving users complete authority over the devices they own.

ArttulOS

ArttulOS is a powerful, open-source Linux distribution built on the rock-solid foundation of Rocky Linux. It delivers the security and reliability of the RHEL ecosystem while serving as the essential OS layer for Cerborus Linux, bridging the gap between traditional servers and modern Android-powered hardware.

System Architecture

ArttulOS holds a unique position. For standard servers, it's a reliable OS. For Cerborus Linux, it's the bridge between hardware and a full Linux experience.

Layer 3: User Applications / Cerborus Linux Run standard Linux applications, containers, or the full Cerberus environment.
Layer 2: ArttulOS (Rocky-based Userspace) The complete and stable operating system.
Layer 1: Linux Kernel / AngusOS HAL The low-level hardware drivers and kernel. For Cerberus, this is the Android Hardware Abstraction Layer provided by AngusOS.
